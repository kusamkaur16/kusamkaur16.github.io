Through out my undergrad, I have taken several interesting and challenging courses. Some of which are:
Network Systems Security
This course focused on teaching attacks on networked systems, tools and techniques used for detection and protection against such attacks. This included the introduction to firewalls and intrusion detection and protection systems. As well as authentication and identification in distributed systems and cryptographic protocols for IP networks.
Large Scale Analytic Systems
In this course, I was introduced to topics dealing with big data analysis. This included the exploration of the map reduce technique using python, hadoop, and pyspark. As well as focusing on both supervised and unsupervised machine learning techniques.
Web based Systems
This course exposed me to various technologies related to web development. This includes exposure to frameworks like angular, vue, and react as well as back end technologies like socket.io and ajax.
Human Computer Interactions
This course explored the fundamental theory and practice of design, implementation, and evaluation of human-computer interfaces. Topics included methods for evaluating interfaces, techniques for prototyping and implementing graphical user interfaces.
Embedded System interfacted
This course offered a review of computer architecture by reexamining microcontrollers and their instruction sets. The focus was on interfacing using common input/output devices, strategies for interrupt handling and exception handling. This interfacing was done using high level (C and C++) and assembly languages. The goal was to identify software and hardware optimizations that would allow us to achieve real time operations for use in real time operating systems.
Project Management
This course explored the analysis of methods, tools, and techniques for software process improvement and software project management. This course helped me get familiar with real world practice of SDLC and techniques that can be used to improve the workflow of the process.
